开始通过Ionic创建!

按照下面的快速步骤然后开始几分钟内创建高质量移动应用。为了更深入的概述，请观看我们的 速成班视频， 它涵盖一切你想要知道的。

安装 Ionic

首先，安装 Node.js。然后安装最新版Cordova 和Ionic 命令行工具. 按照 Android and iOS 平台指南来安装所需平台依赖。

注意：iOS 开发需要 Mac OS X.

{% highlight bash %} $ npm install -g cordova ionic {% endhighlight %}

开始一个项目

用我们的现成的应用中的一个来创建一个项目 Ionic或者一个空白模板重新开始

{% highlight bash %} $ ionic start myApp tabs {% endhighlight %}
$ ionic start myApp blank
$ ionic start myApp tabs
$ ionic start myApp sidemenu

运行它

Ionic应用基于Cordova, 因此我们可以使用 Cordova 工具来编译，测试，和部署我们的应用，但是我们通过ionic工具来提供简单的方式来做同样的事(替换ios为android来编译Android):

{% highlight bash %} $ cd myApp $ ionic platform add ios $ ionic build ios $ ionic emulate ios {% endhighlight %}

耶，就是这么简单。你也许想知道下一步是什么了...
