概述

欢迎！Ionic 是一个强大的HTML5原生应用开发框架帮助你打造原生感觉的移动应用，一切都是web技术比如HTML, CSS, and Javascript。

Ionic主要关注于你应用的外观和体验还有 UI 交互。这就意味着我们不是一个PhoneGap的替代者或者你最喜欢的Javascript框架。相反，Ionic 适合那些简化你应用最大一部分：前后端的项目。我们推荐阅读Ionic 框架适合用在哪里？ 来很好理解框架的目标。

Ionic 目前依赖于AngularJS 为了工作时能发挥出全部潜力。虽然你仍然可以只使用框架的CSS部分,你将会错过强大的交互，手势，动画和其他的东西。将来我们会扩展超出Angular来支持其他的框架。

我们在将来将放出 Cordova/PhoneGap 插件 来提高你应用的兼容性。

命令行工具和种子项目

我们有一个node.js 单元可以用来快速开始一个Ionic项目。我们致力于打造这个工具的功能，现在你就可以快速创建一个通过我们的种子项目 通过做:

{% highlight bash %} $ sudo npm install -g ionic $ ionic start myproject {% endhighlight %}

你也可以克隆种子项目 本身，它给了你需要通过Cordova 和 AngularJS 创建Ionic应用的一切。

包含了什么

Ionic 包含了你应用要用的压缩过的Javascript 和CSS,可选的Sass 文件和JS框架扩展,以及一个伟大的图标包。

CSS/Sass

CSS可以用默认的，但是它也可以被开发者增强。简单的方式你只要加入你的CSS然后覆盖默认属性就行了。为了更强大和灵活，核心是用Sass写的并包含了简单定制的变量和混合。尽管默认手机相近于iOS，我们让CSS处于一个可以易于扩展获得你想要的外观和感觉的版本。

PhoneGap/Cordova/Trigger.io

我们的示例在Cordova 3.4.0上测试的,但是应该在其他任何版本上可工作，因为还没有加入插件。要添加到你的PhoneGap, Cordova, 或者 Trigger.io 项目,只要在你web应用的根目录里引入Ionic CSS 和 JS 文件。

浏览器支持

Ionic 专注于打造 native/hybrid 移动应用多于移动网站。

因此，我们的浏览器支持和Web View Api在相应平台上对移动应用支持的一样。 对于 Ionic 1.0.0-beta.4 "antimony-antelope", 即UIWebView for iOS 6+, 和Android 4+ (一些支持 for 2.3). Windows Phone 和 FirefoxOS 支持在我们的roadmap中。

版权

Ionic 是一个开源框架遵循一个宽容的MIT 协议。这意味着你可以使用Ionic在你个人或者商业项目里免费使用它。MIT 也是很多流行项目使用的协议比如jQuery 和Ruby on Rails。

网站内容在这里(在 ionic-site repo) 是在Apache 2 协议下的。

关于Ionic

Ionic 是由@benjsperry, @adamdbradley, 和 @maxlynch 在 Drifty, 一个独立的自发的软件公司，创建过很多好产品比如Codiqa和Jetstrap里创建的。

Ionic 致力于将移动端的HTML5带入未来。我们想要一个移动框架不仅外观好看和功能良好，而且足够强大到开发者可以创建最神奇的应用。 HTML5的时代已经到来。Ionic 是证明这点的框架
